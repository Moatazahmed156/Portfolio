Hangman-Game

This is a browser-based Hangman game where players try to guess a randomly chosen word by selecting letters. The game includes categories like programming languages, movies, famous people, and countries If the player guesses incorrectly 8 times, the game is over. If the player correctly guesses all letters, they win, and a fireworks animation is displayed.

Task-List

Task Manager is a simple and efficient web application designed to help users manage their daily tasks. Built with HTML, CSS, and JavaScript, this app allows users to easily add tasks, display them in a list, and delete them as needed. The tasks are stored in the browser's localStorage, ensuring that they remain available even after the page is refreshed. With a responsive design and intuitive interface, users can quickly manage their tasks, improving productivity and organization.
